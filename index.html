Subheader #1 - 10 most useful tags in HTML

  1. html — The root element
  2. head — The document head
  3. title — The page title
  4. body — The page’s content
  5. h1 — A section heading
  6. p — A paragraph
  7. a — A link
  8. img — An image
  9. div — A block-level container for content
  10. span — An inline container for content

Subheader #2 - The most important header tags

  1. H1 tag — Acts as an information guide for the search engine bots. It should include a highly searched keyword.
  2. H2 and H3 tags — Commonly used as subheadings.Akin to book chapters, describing the main topics you’ll cover in sections of the article.
  3. H4-H6 tags — H4, H5, and H6 tags are used less frequently.
